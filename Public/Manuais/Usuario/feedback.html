Realizando um feedback

Você pode fazer feedbacks apenas após fazer uma busca por transporte público

Para fazer um feedback, primeiro faça uma busca, e clique no link do modal, após fazer seu comentario na tela de feedback,
clique no box abaixo caso tenha gostado do serviço, como apresentado abaixo, esse feedback é importante para sabermos se esta satisfeito com os serviço do vai card ou não.
